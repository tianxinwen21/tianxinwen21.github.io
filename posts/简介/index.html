Contents
  1. hugo //生成public文件夹
  2. cd public //进入public文件夹
  3. git init // 初始化public文件夹为本地git仓库
  4. git add . // 将全部文件提交
  5. git commit -m “first commit” // 提交信息
  6. git branch -M main // 选择GitHub 的main分支
  7. git remote add origin https://github.com/tianxinwen21/tianxinwen21.github.io.git //选择你的github仓库,一定要是user.github.io
  8. git pull origin main // 拉取你云端仓库的READMD.md文件(如果没有勾选,就不需要这一步 (拉取可能会有报错信息,去搜索解决,所以建议不勾选创建READMD文件) 9.git push -u origin main //提交到GitHub

写博客并且上传

$ hugo new posts/name.md //然后在vscode中写博客内容 $ hugo $ cd public $ git add . $ git commit -m “tiaojiao” $ git push

安装皮肤

$ cd themes //进入存放主题的文件夹 $ git clone 皮肤地址 //将皮肤地址克隆下来
